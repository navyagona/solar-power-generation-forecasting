# Solar Power Prediction using Ensemble Machine Learning

## Overview
This project predicts solar power generation using an ensemble machine learning model trained on historical power and weather data. The goal is to provide accurate and stable power predictions through a simple and interactive web application.

## Features
- Ensemble regression model for improved accuracy
- Weather and time-based feature usage
- Streamlit-based user interface
- Simple and clean project structure

Note: The trained model file is not uploaded due to GitHub file size limits.
It can be generated by running the training script or shared on request.
All the datasets used can be found inside the model folder.
